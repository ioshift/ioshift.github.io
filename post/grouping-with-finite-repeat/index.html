Grouping with Finite Repeat

2023-11-24 2 min read # solved

Problem

将一群人分为4个人的组，任意两个组中有且仅有2人是重复的，且没有2人在所有组里，最多能分出几组？

Solution

反证法，假设组数 ≥8\ge 8≥8

首先易证同一个两个元素的 pair 最多出现在 333 个组里 [1]

而且如果有一个 pair 在 333 个组里，那么所有的组均包含这个 pair 的至少一个元素 [2]

我们知道一个组包含 666 个 pair，于是，根据抽屉原理，至少有一个 pair 在 333 个组里，不失一般性，设这 333 个组为 1234,1256,12781234,1256,12781234,1256,1278

考虑组 123412341234，新的组不能含有 121212 或 343434，而对于剩下的 444 个 pair，根据抽屉原理，必有一个 pair 包含在 333 个组中，不妨设其为 131313（现在我们已经构造出了 555 个组）

现在新的组只能含有 141414 或 232323，而显然它们无法构造出 333 个新组，于是假设不成立 [3]

组数等于 777 的情况是容易构造的（如1234,1256,1278,1358,1367,1457,14681234,1256,1278,1358,1367,1457,14681234,1256,1278,1358,1367,1457,1468）

于是，组数最大为 777

  1. 否则，例如存在 1234,1256,1278,129A1234,1256,1278,129\rm{A}1234,1256,1278,129A，那么显然无法构造出（不包含 121212 的）第 555 个组 ↩︎

  2. 例如存在 1234,1256,12781234,1256,12781234,1256,1278，那么如果有一个组没有包含 121212，那么它必须同时包含 34,56,7834,56,7834,56,78，这显然不成立 ↩︎

  3. 首先这 333 个新组不能含有同一个 pair，否则那个 pair 在一共 444 个组里；于是，由抽屉原理，必有一个 pair 包含在 222 个新组中 （如 141414），但这样的话就不能包含 232323 了（因为 141414 被包含在一共 333 个组中），于是不成立 ↩︎
