Compactness & Limit Points

2023-11-23 2 min read # solved

Problem

Construct a compact set KKK with its infinite subset EEE such that there exists an limit point of EEE which isn't contained in KKK.

Solution

Actually trivial üòÉ

We can easily prove that such set doesn't exist.

For all limit point of EEE, it means all its neighborhood contains infinite points in EEE, so it must contains infinite points in KKK, and thus it's a limit point of KKK. Then since KKK is a compact set and thus a closed set, all limit points of KKK must be contained in KKK, which implies all limit points of EEE must be contained in KKK.

Comment
